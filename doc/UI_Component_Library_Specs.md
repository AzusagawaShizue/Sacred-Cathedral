# UI Component Library Specifications

## 1. Buttons

### Primary Button (Gold)
*   **Background**: Linear Gradient (`#D4AF37` to `#C5A028`)
*   **Text**: `#0F172A` (Midnight Blue) - **Bold**
*   **Border**: None or 1px Solid `#FAF7F0` (Inner Ring)
*   **Hover**: Brightness 110%, Shadow `0 0 15px rgba(212, 175, 55, 0.6)`
*   **Active**: Scale 0.98

### Secondary Button (Glass)
*   **Background**: `rgba(255, 255, 255, 0.1)`
*   **Text**: `#EEDCB3` (Soft Gold)
*   **Border**: 1px Solid `#EEDCB3`
*   **Hover**: Background `rgba(255, 255, 255, 0.2)`

## 2. Cards (The "Relic" Container)
*   **Background**: Dark Glass (`.glass-sacred`)
*   **Border**: 1px Solid Gradient (Gold to Transparent)
*   **Padding**: `24px` (6 units)
*   **Corner Radius**: `16px`

## 3. Modals
*   **Overlay**: Black at 80% opacity, backdrop blur 4px.
*   **Content**:
    *   Background: Deep Midnight Blue with "Marble" or "Noise" texture overlay.
    *   Border: Double Gold Border (Baroque Style).
    *   Animation: Scale In + Fade In.

## 4. Inputs
*   **Background**: `rgba(0, 0, 0, 0.3)`
*   **Text**: `#FAF7F0`
*   **Border**: Bottom Border 2px `#717182` (Muted), focus turns to `#D4AF37` (Gold).
*   **Placeholder**: `#717182`
